# E-commerce Vue Application

A modern e-commerce application built with Vue 3.

## Project Setup

```bash
npm install
```

### Compile and Hot-Reload for Development

```bash
npm run dev
```

### Compile and Minify for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- Vue 3
- Vue Router
- Pinia (State Management)
- Axios (HTTP Client)
- Vite (Build Tool)
